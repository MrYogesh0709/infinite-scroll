
# Infinite Scroll

Simple Weather api usecase for get Weather of current location and with search of city.


## Demo

https://react-infinite-scroll-vite.netlify.app
## Installation

Install my-project with npm

```bash
  npm install 
  cd infinite-scroll
  npm run dev
```

## Features

- react + vite
- infinite-scroll
- react-query
